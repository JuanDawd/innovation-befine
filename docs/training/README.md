# Guías de uso — BeFine

Guías de capacitación interna, una por rol. Cada guía cubre las tareas diarias más frecuentes y puede leerse en menos de 5 minutos.

| Rol                    | Archivo                              | Tareas principales                                             |
| ---------------------- | ------------------------------------ | -------------------------------------------------------------- |
| Administrador / Cajero | [cashier-admin.md](cashier-admin.md) | Abrir/cerrar día, cobrar, liquidar nómina, gestionar catálogo  |
| Estilista              | [stylist.md](stylist.md)             | Registrar servicios, marcar para pago, ver ganancias           |
| Confeccionadora        | [clothier.md](clothier.md)           | Tomar piezas, marcar como hechas, ver ganancias                |
| Secretaria             | [secretary.md](secretary.md)         | Agendar citas, registrar clientes, crear lotes, aprobar piezas |

> Para preguntas técnicas o problemas con la app, contactar al administrador del sistema.
